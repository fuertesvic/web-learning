  • Aprenentatge de Desenvolupament Web

Tota la web és un espai d'experimentació, experiència i joc, el qual faig servir per aprendre tecnologies web.
En aquesta pàgina en concret vaig documentant totes (o la majoria) d'eines que vaig aprenent, per anar familiaritzant-me.

Per ara, les tecnologies que estic aprenent son les més bàsiques: HTML i CSS

Llista de tags HTML útlils

    Organització de pàgina

  1. header: Defineix una capçalera per la pàgina.
  2. footer: Defineix una secció a baix de tot de la pàgina.
  3. nav: Crea un menú de navegació, com el menú d'aquesta web: pàgina principal - projectes - contactes... etc.
  4. main: Serveix per declarar la part central del teu document; tot el que no forma part de header o footer.
  5. h1,h2,h3: Seccions. Cada número successiu és un subnivell més de secció p.ex h1: Secció General, h2 secció més específica
  6. a: Ve d'anchor i serveix per crear un enllaç a una pàgina web, un arxiu, o una ubicació dins d'una mateixa pàgina. Molt utilitzat amb l'argument href. Per exemple, per posar un enllaç a google.com, la sintaxi és <a href="google.com">
  7. section: Serveix per definir una secció de continguts. Amb l'argument id, després es pot referenciar.
    Per exemple, per crear una secció per html i després referenciar-la, la sintaxi és <section id="section1"> i després, <a href=#section1>

    8. Creació de text bàsica

  8. p: Tot el que va entre mig del tag "p" (ve de paragraph) té una línea en blanc abans i després.
    Si no hi ha la "p", per moltes línies blanques que hi hagi al codi, el text anirà seguit.
  9. br: L'etiqueta br, que ve de break i es fa servir per saltar a la següent línia.
  10. b: De bold, tot el que va entre aquesta etiqueta estarà en negreta
  11. i: D' italics, tot el que va entre aquesta etiqueta estarà en cursiva
  12. ol: Llista nombrada (com aquesta mateixa)
  13. ul: Llista no nombrada (bullet points)
  14. li: Ve de list item, serveix per declarar un punt/element d'una llista (p.ex: 1. Element, o ·Element)
  15. Argument class: Serveix per definir l'estil d'un element. Al CSS s'especifica la classe i es defineix l'estil, com per exemple color, tipus de font, color de fons, alineació, etc.

    16. Elements que s'han de posar (o és molt recomanable) a l'inici d'un document HTML5

  16. <DOCTYPE html> Això li comunica al navegador que el document és de tipus HTML5
  17. html: L'arrel de l'html. S'ha de posar al princi i al final.
  18. headS'utilitza per definir metadades del document, com per exemple:
    • <meta charset="UTF-8">: Defineix el tipus d'encoding
    • <link rel="stylesheet" href="style.css">: Enllaça el document HTML amb un css anomenat style.css (nom estàndard) on es defineix l'estil (visual) del document.
    • <title>: Serveix per definir el títol de la pàgina (el que surt a la pestanya del navegador, en aquest cas "Aprenentatge Web").

    19. Afegir elements al document

  19. img: Serveix per afegir images. Necessita el paràmetre "src" de source indicant la ruta d'on prové la imatge

    21. Crea un formulari on l'usuari introdueix dades
  21. label: descriu Inputs
  22. input: defineix un camp on l'usuari ha d'introduir dades. Pot ser de diferents tipus; definits amb l'argument type.
    Per exemple, per definir un input de text, la sintaxi és <input type="text">
